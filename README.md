# 🏛️ AcehEdu Premium - Interactive History Module

AcehEdu adalah platform edukasi interaktif yang menyajikan sejarah **Kesultanan Aceh Darussalam** dengan tampilan modern dan imersif.

### ✨ Fitur Unggulan
* **Immersive UI:** Tema gelap dengan aksen emas premium.
* **Canvas Animation:** Efek bintang jatuh dan starfield dinamis menggunakan JavaScript.
* **Micro-Interactions:** Efek 3D Tilt dan Magnetic Hover pada kartu materi.
* **Mobile Ready:** Responsif untuk semua perangkat.

### 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Variables), JavaScript (ES6), Bootstrap 5.
* **Backend:** PHP (Ready for Database integration).
* **Library:** Google Fonts (Cinzel & Inter).

### 🚀 Cara Penggunaan
1. Clone repository ini.
2. Jalankan di server lokal (XAMPP/Laragon).
3. Buka `index.php` melalui browser.
